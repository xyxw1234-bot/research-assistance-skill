# Changelog

## 0.2.1 — 2026-07-22

- 规范名称统一为“科研辅助 Skill”。
- Skill 标识、仓库名称、README、安装说明、Release 文案和测试报告统一为 `research-assistance-skill`。
- 清理旧产品名、旧仓库名、旧标签、旧 Release 与旧提交历史，避免公开页面残留混乱称呼。

## 0.2.0 — 2026-07-22

### 能力强化

- 增加资料与文献智能：检索计划、来源卡、证据台账、作者/团队公开实践分析。
- 增加方向扩展与文献综述：方向树、近邻工作矩阵、综述结构和研究缺口约束。
- 强化研究实施：变量、对照、失败判据、最小试点、风险和复现要求。
- 增加数据规范与结果解释：数据字典、质量规则、分析计划、结果表述边界。
- 强化论文写作支持：章节—证据映射、强论断检查、审稿预检。
- 明确长期项目档案、导师反馈、跨会话待办和长期决策日志不在本 Skill 范围。
