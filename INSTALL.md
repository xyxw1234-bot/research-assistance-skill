# 安装科研辅助能力包（Skill）

**当前正式版本：v0.2.3。**

## 给使用者

把下面的原始链接发送给您的 Hermes 或兼容智能体，并说明：

> 请安装并启用科研辅助能力包（Skill）。安装成功后，请立即完整发送该 Skill 规定的欢迎介绍与使用建议。

在 Hermes CLI 中执行：

```bash
hermes skills install https://raw.githubusercontent.com/xyxw1234-bot/research-assistance-skill/main/SKILL.md
```

安装或更新成功后，智能体必须自动完整推送欢迎介绍；您无需重复提出这一要求，随后可直接提出研究任务。安装后也可按您的需要增删、改名、重排或扩展能力模块与输出模板。

## 首次使用建议

选择当前最需要的一类工作，并给出可读材料或明确范围：

1. **文献检索与证据整理**：主题、学科、时间范围、已有论文或链接；
2. **选题拓展与文献综述**：现有选题、目标人群或场景、已知近邻工作；
3. **研究设计与试点方案**：研究问题、资源约束、可用数据、预期比较对象；
4. **数据规范与结果解读**：数据说明、字段、研究问题、已完成或计划的分析；
5. **论文写作**：草稿、目标章节、需要核验的主张和现有来源。

材料不足时，智能体应先给检索策略、字段模板或待核验清单，不应编造结论。

## 更新策略

安装入口使用 `main` 分支，便于新安装者获得后续更新；版本标签和 GitHub Release 用于审计与回退。
