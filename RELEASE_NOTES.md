# 科研辅助能力包（Skill）v0.2.2

本版本将面向用户的展示名称调整为“科研辅助能力包（Skill）”，并增加安装或更新成功后必须自动完整推送的能力介绍与使用建议。

## 科研辅助能力包（Skill）可以协助的工作

- 资料、论文、网页、代码和用户文件的检索规划、筛选与证据整理；
- 作者、团队和开源项目的公开实践分析；
- 研究方向扩展、近邻工作比较和文献综述结构；
- 研究问题、变量、对照、最小试点、风险与复现设计；
- 数据字典、质量规则、分析计划、图表计划和结果解释边界；
- 论文主张—证据检查、段落修订和审稿预检。

## 重要边界

- 不伪造文献、作者实践、数据、实验结果、统计结论或新颖性；
- 工具或材料不足时，输出待核验策略，不冒充已完成检索或分析；
- 不自动执行下载、训练、构建、删除、投稿或外部发布；
- 长期项目档案、导师反馈和任务追踪不在本 Skill 范围，未来由独立的科研项目管理 Skill 承担。
