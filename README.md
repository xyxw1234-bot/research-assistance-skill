# 科研辅助能力包（Skill）

面向硕士、博士、研究生和科研团队的 Hermes Skill。它以证据为中心，协助完成文献检索与证据整理、选题拓展与文献综述、研究设计与试点方案、数据规范与结果解读、论文写作与审稿预检。

面向用户的展示名称为“科研辅助能力包（Skill）”；仓库和安装标识保持为 `research-assistance-skill`。

**当前正式版本：v0.2.3。**

## 能力范围

| 模块 | 可协助完成的工作 |
|---|---|
| 文献检索与证据整理 | 设计检索策略、筛选来源、建立文献/证据卡、分析作者或团队的公开实践 |
| 选题拓展与文献综述 | 形成方向树、近邻工作矩阵、概念框架、综述结构和研究缺口 |
| 研究设计与试点方案 | 明确变量、对照、指标、失败判据、最小试点、复现和风险清单 |
| 数据规范与结果解读 | 设计数据字典、质量规则、分析计划、图表计划和结论边界；有工具和明确授权时协助实际分析 |
| 论文与审稿 | 建立章节故事线、主张—证据表、段落修订、强论断检查和审稿预检 |

## 不包含的工作

科研辅助能力包（Skill）不保存长期项目档案、导师会议纪要、跨会话待办、版本历史或决策日志。这些需要独立的、带持久化存储的“科研项目管理 Skill”。

## 安装

- [仓库说明页](https://github.com/xyxw1234-bot/research-assistance-skill)
- [智能体直接安装入口（SKILL.md）](https://raw.githubusercontent.com/xyxw1234-bot/research-assistance-skill/main/SKILL.md)

在 Hermes CLI 中执行：

```bash
hermes skills install https://raw.githubusercontent.com/xyxw1234-bot/research-assistance-skill/main/SKILL.md
```

安装或更新成功后，智能体会**自动完整推送能力介绍与使用建议**；无需记忆命令，您可直接按其中示例开始研究任务。

## 可按您的需要调整

五个能力模块是默认起点，不是固定套餐。您可以随时要求智能体：

- 保留、删减、改名、重排或新增能力模块；
- 换成符合您所在学科、课题组或研究阶段的工作流程；
- 调整检索表、证据卡、研究设计表、数据模板、写作模板和输出语言。

例如：“保留文献与写作模块，暂时不需要数据分析”“把研究设计模块改成教育实验设计”“按我们课题组的开题报告结构输出”。

## 推荐提问方式

- “请为我的研究主题建立中英文检索策略、筛选规则和文献证据卡模板。”
- “请根据这些论文和作者主页，整理作者/团队的公开研究实践，不要做私人推断。”
- “基于已核验文献，扩展我的研究方向并形成综述大纲。”
- “请把这个选题设计成最小可行试点，包括变量、对照、指标和失败判据。”
- “根据这份数据字段说明，生成数据字典、质检规则和统计分析计划。”
- “请检查这段论文是否每项强论断都有对应证据，并把不足之处标为待补。”

## 工具与真实边界

Skill 负责工作流、证据纪律和输出结构。实际能力取决于接收智能体可用的工具：

- 有网页、论文解析或文件读取工具时，可以真实读取来源并核验；
- 有数据/代码工具且研究者明确授权时，可以协助处理实际数据；
- 工具或材料不足时，只给检索策略、模板、分析计划和待核验清单；
- 不会伪造已阅读文献、作者实践、数据结果或统计结论。

## 学术诚信与安全

- 不伪造参考文献、DOI、作者、实验结果、数据集、基线和统计结论；
- 不把用户材料或 AI 推测写成已验证事实；
- 不自动执行下载、爬取、训练、构建、删除、投稿或外部发布；
- 研究方向、伦理、数据真实性、署名和投稿由研究者负责。
